## Requirement

- Requirement 설치
```bash
$ git clone https://github.com/nlpproject-2019/NLP_project
$ cd NLP_project
$ pip install -r requirement.txt
```
- 실행 명령어
```bash
$ cd NLP_project/src
$ python3 Main.py
```
